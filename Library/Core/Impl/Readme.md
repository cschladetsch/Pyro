# Core.Impl
Implementation of the Core library for *Pyro*.

You can think of this as being equivalent to the `System` dll for .Net.

Contains definitions for, among other things:
* IRegistry
* IObject
* IClass
* ITree
* ICallable
* IReflected
* Identifier
* Process
